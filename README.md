# React Todo List

Учебный проект — приложение для управления задачами (todo list) на React.

## Описание проекта
Этот проект представляет собой учебное React-приложение для управления списком задач (todo list). Его основная цель — помочь новичкам понять основные концепции React, такие как управление состоянием, обработка событий и создание компонент.

Приложение позволяет пользователю в интерактивном режиме создавать новые задачи, просматривать список уже добавленных задач и удалять выполненные или ненужные. Всё реализовано в виде простого и понятного интерфейса, что делает его идеальным для изучения основ React и практики работы с компонентами.

В процессе работы пользователь может добавлять задачи через специальное поле ввода, которые тут же отображаются в списке. Каждая задача сопровождается кнопкой для удаления, что позволяет легко управлять списком. Все данные хранятся в состоянии компонента, что демонстрирует работу с реактивностью React и обновлением интерфейса при изменениях.

Проект предназначен для начинающих разработчиков, которые только начинают осваивать React, а также для тех, кто хочет закрепить навыки работы с компонентами, состоянием и обработчиками событий. В будущем проект можно расширять, добавляя новые функции, такие как возможность отмечать задачи как выполненные, сохранять список в localStorage или интегрировать с бэкендом.

## Основные функции

- Добавление новых задач в список
- Просмотр всех текущих задач
- Удаление выбранных задач
- Простая и понятная реализация

## Установка и запуск

1. Создайте новый проект React или используйте существующий:
   
   ```bash
   npx create-react-app my-todo-app
   cd my-todo-app
   ```

2. Создайте файл `TodoApp.jsx` в папке `src/` и вставьте туда код компонента.

3. Импортируйте компонент `TodoApp` в файл `src/App.js` и используйте его:

   ```jsx
   import React from 'react';
   import TodoApp from './TodoApp';

   function App() {
     return (
       <div className="App">
         <TodoApp />
       </div>
     );
   }

   export default App;
   ```

4. Запустите проект командой:

   ```bash
   npm start
   ```

Проект откроется в браузере по адресу `http://localhost:3000`, где вы сможете взаимодействовать с todo-листом.
